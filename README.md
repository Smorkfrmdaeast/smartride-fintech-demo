# SmartRide NG – Mobile Transportation & Fintech Booking App

SmartRide NG is a **mobile-first fintech ride booking platform** built with **React**.  
It allows users in Nigeria to **book Keke (tricycles), Okada (motorbikes), Cabs, and Buses**, pay via a simulated wallet, track drivers, and view booking history. Designed as an **investor-ready MVP**, showcasing modern UI, fintech flow, and scalable architecture.

---

## **Demo Features**

- **Vehicle Selection**: Keke, Okada (bike), Cab, Bus  
- **Fare Calculation**: Distance-based, displayed in ₦  
- **Wallet Simulation**: Deduct fares from user wallet  
- **Booking History**: View past rides  
- **Fake Driver Tracking UI**: Animated driver dot on map  
- **Driver Profile Card**: Name, rating, vehicle type  
- **Admin Dashboard Simulation**: Total rides, total revenue, active drivers  
- **Dark/Light Mode Toggle**: Switch UI themes  
- **Mobile-First Design**: Responsive and colorful UI

## **How to Run**

1. Clone or download this repo.  
2. Open `index.html` in a browser (works **directly without build tools**).  
3. Ensure `images/` folder is in the same directory as `index.html`.  

---

## **Technical Stack**

- **Frontend**: React (CDN UMD via Babel)  
- **State Management**: React Hooks (`useState`, `useEffect`)  
- **Styling**: CSS (responsive, mobile-first, dark/light mode)  
- **Animation**: CSS keyframes for driver tracking  
- **Hosting**: GitHub Pages ready
